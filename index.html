Мои работы

Гипервизор Sand

Гипервизор 2-ого типа, основывающийся на технологии аппаратной виртуализации Intel VT-x. Позволяет запускать гостевой код в изолированном окружении. Использует механизм EPT для отображения гостевой памяти. Гипервизор реализован в виде модуля ядра Linux.

Криптографический модуль CryptoPipe

Модуль ядра Linux, предоставляющий несколько каналов шифрования/дешифрования данных. Основан на алгоритме Blowfish, реализующим блочное симметричное шифрование с переменной длиной ключа. Доступ к каналу осуществляется через специализированный файл устройства /dev/cpipe.

Система автоматизированного синтеза программных прошивок

Представляет из себя программный комплекс для создания прошивок приборов с кафедры К3 МФ МГТУ. Ключевой особенностью решения является его модульность: конкретные прошивки для приборов собираются из готового набора программных модулей, которые требуют лишь минимальной доработки для совместного использования.

Комплект лабораторных работ по сиcтемному программированию для x86

Включает в себя несколько лабораторных лабот, посвященных рассмотрению основных аспектов процессоров Intel x86: переход в защищенный режим, обработка исключений и аппаратных прерываний, страничное преобразование адресов, многозадачность. Лабораторные работы предназначены для курса "организация ЭВМ".

Комплект лабораторных работ по периферийным устройствам

Включает в себя лабораторные работы, посвященные изучению различных периферийных устройств (терминал, клавиатура, мышь) и их интерфейсов (JTAG UART, RS232, irDA, PS/2). Лабораторные работы предназначены для курса "Периферийные устройства ЭВМ".
